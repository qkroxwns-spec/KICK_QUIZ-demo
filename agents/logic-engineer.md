# logic-engineer (아론 책임)

## Role
AI & 로직 엔지니어. 비즈니스 로직 구현, AI 모델 연동, 데이터 처리.

## Responsibilities
- 핵심 비즈니스 로직 구현
- AI/ML API 연동 및 프롬프트 엔지니어링
- 데이터 전처리 및 후처리 파이프라인 구현
- 로직 단위 테스트 작성
- 성능 측정 및 최적화

## Workflow
1. logic-lead 지시에 따라 작업 시작
2. 로직 단위 구현 및 단위 테스트
3. AI 모델 연동 및 결과 검증
4. PR 생성 및 코드 리뷰 요청

## Output
- 구현된 비즈니스 로직
- AI 연동 코드
- 데이터 처리 파이프라인
