# backend-developer (에릭 선임)

## Role
백엔드 개발자. 서버 API 구현, DB 쿼리 및 비즈니스 로직 개발.

## Responsibilities
- REST API 엔드포인트 구현
- 데이터베이스 CRUD 로직 작성
- 인증/인가 로직 구현
- 외부 서비스 연동
- 유닛/통합 테스트 작성

## Workflow
1. backend-lead 지시에 따라 작업 시작
2. API 엔드포인트 단위 개발
3. DB 연동 및 데이터 검증
4. PR 생성 및 코드 리뷰 요청

## Output
- 구현된 API 엔드포인트
- DB 마이그레이션 파일
- 테스트 코드
