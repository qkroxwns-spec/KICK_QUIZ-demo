# 컴플라이언스 체크리스트 — 토스 미니앱 심사 대응

## 1. 사행성 심사 대응

| 체크 항목 | 상태 | 근거 |
|-----------|------|------|
| 참여비 0P (유저 재화 소모 없음) | ✅ | 퀴즈/승부차기 무료 참여 |
| 결과 무관 리워드 | ✅ | 승부차기 골/실패 모두 100P |
| 일일 참여 제한 | ✅ | 퀴즈 1회, 승부차기 1회, 박스 1회 |
| 리워드 액수 고정 | ✅ | 정답=100P, 오답=50P 차이만 (참여 중심) |
| 현금 환전 불가 | ✅ | P는 간식박스 오픈 용도, 현금 전환 불가 |
| 확률 고지 | ✅ | 간식박스 확률 테이블 공개 |

## 2. 저작권/상표권 대응

| 체크 항목 | 상태 | 근거 |
|-----------|------|------|
| 구단 엠블럼 미사용 | ✅ | 공식 로고 이미지 사용 안 함 |
| 컬러 배지 + 약어만 사용 | ✅ | 예: 🔴 ARS (텍스트만) |
| 대회 공식 명칭 사용 | ✅ | "EPL", "월드컵" 일반 명칭 |
| 이미지 출처 표기 | ✅ | 해설에 Wikipedia 등 출처 표기 |
| 선수 초상권 | ✅ | 이미지 퀴즈 없음 (텍스트만) |

## 3. 데이터 보호

| 체크 항목 | 상태 |
|-----------|------|
| 개인정보 수집 최소화 | ✅ 닉네임만 (카카오 연동 시) |
| 로컬 스토리지 사용 | ✅ 서버 DB 최소화 |
| 암호화 | ✅ (필요 시) |
