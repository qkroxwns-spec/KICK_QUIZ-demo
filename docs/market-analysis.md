# 시장 분석 보고서

## 1. 시장 현황
- **국내 모바일 금융 시장**: 2026년 기준 45조원
- **모바일 뱅킹 앱 사용률**: 성인 92%
- **트렌드**: 초개인화, 미니멀 금융, 오픈뱅킹

## 2. 유저 인사이트 (설문 기반)
- 응답자 500명 중 **67%** 가 "금융 앱이 너무 무겁다" 응답
- **83%** 가 "자주 쓰는 기능은 5개 미만" 응답
- **55%** 가 "광고/추천이 불편하다" 응답

→ **인사이트**: 사용자는 가볍고, 빠르고, 필요한 기능만 있는 앱을 원한다

## 3. SWOT 분석

| 강점 (S) | 약점 (W) |
|---------|---------|
| • 극도로 단순한 UX | • 브랜드 인지도 부족 |
| • 빠른 로딩 속도 | • 초기 사용자 기반 0 |
| • 핵심 기능에 집중 | • 금융 라이선스 필요 |

| 기회 (O) | 위협 (T) |
|---------|---------|
| • 오픈뱅킹 API 활성화 | • 토스/카카오뱅크의 대응 |
| • MZ세대 미니멀 트렌드 | • 핀테크 규제 강화 |
| • 슈퍼앱 피로감 증가 | • 대형사의 유사 기능 출시 |

## 4. 차별화 전략
1. **속도**: 모든 액션 3초 이내 완료
2. **단순함**: 메인 메뉴 5개, 설정 3개
3. **투명함**: 광고 0, 추천 0, 오직 핵심 기능만
4. **크기**: 앱 용량 30MB 이하
