# 광고 연동 계획

## 1. 광고 슬롯 위치
- **퀴즈 내**: 초급2문항 → [광고] → 고급2문항 사이
- **크기**: 320×100px (모바일 배너)
- **노출 타이밍**: 퀴즈 진행 중 자연스러운 쉬는 타이밍

## 2. 연동 옵션

| 옵션 | 장점 | 단점 | 결정 |
|------|------|------|------|
| **Toss 광고** | Toss 생태계, 정책 적합 | Toss 광고주 네트워크 확인 필요 | ⬅️ 추천 |
| **Google AdSense** | 쉬운 연동, 높은 수익 | Toss 웹뷰 정책 위반 가능성 | 비추 |
| **카카오 AdFit** | 국내 최적화 | 최소 트래픽 조건 | Post-MVP |
| **자체 배너** | 제어 용이 | 수익 0원 | MVP 유지 |

## 3. Toss 광고 연동 (추천)
- Toss 광고 SDK: `toss-ad.js` (Toss 개발자센터에서 배포 예정)
- 슬롯 ID 발급 필요
- 예시 (준비):
```html
<div id="ad-slot" data-toss-ad="SLOT_ID"></div>
<script src="https://static.toss.dev/toss-ad/latest.js"></script>
```

## 4. MVP (현재)
- 광고 슬롯 자리 유지 (더미 텍스트)
- `app/index.html` 광고 영역 CSS/HTML 완비
- 실 광고주 확정시 `AD_SLOT_ID`만 교체
