# 기능 명세서 v3 — KICK QUIZ (회의 반영)

## [1] 홈 탭 — 일일 루틴 챌린지

### F1.1 오늘의 퀴즈 (4문항)
- 초급 2 + 고급 2 셔플, O/X 스와이프
- F5 치트 방지: 로컬 스토리지 세션 동기화
- 진행: progress bar 1/4 ~ 4/4
- 리워드: 정답 100P, 오답 50P (참여 중심)
- 완료 후: "오늘의 축구 역사" TMI 노출

### F1.2 오늘의 축구 역사 (v3 추가)
- 퀴즈 완료 시 하단에 1개 표시
- 예: "1998년 7월 22일, 프랑스 월드컵에서 이란은 미국을 2-1로 꺾었다."
- 단순 정보 제공, 무보상
- 소스: Wikipedia "오늘의 축구" 페이지 참고

### F1.3 승부차기
- 50% 확률, 결과 무관 고정 100P
- 버튼 하나: "찬다!" → 애니메이션 0.5s
- 1일 1회

### F1.4 스트릭 현황
- 상단 표시: "🔥 7일 연속"
- 스트릭 달성 시 Toast 팝업

## [2] 응원 탭 — 주간 매치 + MVP

### F2.1 매치 캘린더
- 네이버 스포츠 크롤링 → 주간 일정
- 표시: 날짜/시간/컬러배지/팀약어/스코어/상태
- 구단→컬러 Dictionary 매핑

### F2.2 유니폼 컬러 배지
- 공식 엠블럼 사용 금지
- 대신: 구단 컬러 2색 블록 + 3자 약어
- 사전 데이터: 30개 구단 (EPL 20 + 분데스 5 + 라리가 5)

### F2.3 응원하기
- 매치 카드 우측 "응원👍" 버튼
- 1경기 1회, 10P

### F2.4 MVP 맞히기 (v3 추가)
- **조건**: 경기 종료(양 팀 득점 있음) 후 2시간 내 투표 가능
- **선지**: 홈팀 선수 2명 / 원정팀 선수 2명 (크롤링 or 수동)
- **결과**: 경기 공식 MOM 발표 후 정답자에 20P
- **사행성 회피**: 참여비 0P, 결과 무관 참여 자체 보상 없음 (정답만 보상)

## [3] 간식박스 탭 — 리워드 정산

### F3.1 P 시스템
| 획득처 | 일일 획득량 | 비고 |
|--------|------------|------|
| 퀴즈 | 200~400P | 정답/오답 차이 |
| 승부차기 | 100P | 결과 무관 |
| 스트릭 보너스 | 가변 | 7일/30일/100일 |
| 응원/MVP | 10~20P | 소소한 보너스 |
| 친구 초대 | 300P | 일회성 |

### F3.2 간식박스 오픈
조건: 500P 보유 + 1일 1회

확률 테이블 (동일):

| 등급 | 확률 | 당첨 | 실비용 |
|------|------|------|--------|
| 1등 | 0.01% | 편의점 3,000권 | 3,000원 |
| 2등 | 0.05% | 육개장 기프티콘 | 1,000원 |
| 3등 | 0.30% | 츄파춥스 교환권 | 500원 |
| 디지털 | 99.64% | VAR권/복구P | 0원 |

### F3.3 디지털 아이템 (v3)
- **VAR권**: 퀴즈 1문제 스킵 (오답 면제) — 보유 중 자동 적용
- **복구 포인트**: 100P 스트릭 유지권 (자정 전 사용 시 오늘 스트릭 유지)
- **3개 이상 중복**: P로 자동 환전 (50P)

## [5] 친구 초대 (v3 추가)

### F5.1 초대코드 생성
- 진입: 설정 > 친구 초대
- 생성: SHA256(닉네임+날짜) 6자리 앞자리
- 공유: "킥퀴즈 함께해요! 내 코드: KQ-XXXX"

### F5.2 초대 입력 (신규 유저)
- 첫 실행 시 "초대 코드가 있나요?" 입력창
- 코드 검증 (클라이언트 해시 검사, 서버 없음)
- 성공 시 300P 자동 지급 (로컬 스토리지)

### F5.3 초대자 보상
- 피초대자가 코드 입력 시 초대자에게 300P
- 오픈카톡 증빙 방식: 피초대자가 초대자 닉네임 전송
- 대표님이 수동 확인 후 P 적립
