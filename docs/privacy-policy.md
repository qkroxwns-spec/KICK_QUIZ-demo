# 개인정보처리방침 — KICK QUIZ

**최종 업데이트**: 2026.07.22

## 1. 수집하는 정보
KICK QUIZ는 **어떤 개인정보도 수집하지 않습니다**.
- 모든 데이터는 사용자 브라우저의 `localStorage`에만 저장됩니다
- 서버로 전송되는 데이터가 없습니다 (MVP 기준)
- Toss 광고 연동시 광고 식별자만 제3자(광고사)에 전달될 수 있음

## 2. 저장하는 정보 (로컬 전용)
| 데이터 | 용도 | 저장 위치 |
|--------|------|----------|
| 포인트, 스트릭, 정답률 | 게임 진행 | localStorage |
| 획득 아이템 (복구권, 부스터 등) | 인벤토리 | localStorage |
| 경기 매치 데이터 | 응원탭 표시 | localStorage |
| 응원팀 선택 | 팬팀 설정 | localStorage |

위 데이터는 사용자가 브라우저 데이터를 삭제하면 함께 삭제됩니다.

## 3. 제3자 공유
- 현재 MVP에서는 제3자에게 데이터를 제공하지 않습니다
- 추후 광고 연동시: Toss 광고 SDK 또는 Google AdSense 정책에 따름

## 4. 사용자 권리
- 모든 데이터는 브라우저 `localStorage.clear()` 또는 설정 → 방문 기록 삭제로 즉시 파기 가능
- 별도 회원가입 없음 → 계정 삭제 요청 불필요
