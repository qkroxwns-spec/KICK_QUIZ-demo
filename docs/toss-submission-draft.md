# Toss 미니앱 심사 제출문 — KICK QUIZ

## 1. 앱 기본 정보

| 항목 | 내용 |
|------|------|
| **앱 이름** | KICK QUIZ (킥퀴즈) |
| **카테고리** | 스포츠 / 퀴즈 |
| **지원 버전** | v1.0.0 (MVP) |
| **타겟 플랫폼** | Toss 미니앱 (웹뷰) |
| **배포 URL** | `https://kick-quiz.vercel.app` |

### 심사용 설명

> KICK QUIZ는 축구 팬을 위한 일일 퀴즈 미니앱입니다. 매일 4개의 OX 퀴즈를 풀고 스트릭(연속 출첵)을 쌓아 포인트를 모으고, 모은 포인트로 '간식박스'를 오픈해 실물 간식을 받을 수 있습니다. K리그 응원하기와 MVP 맞히기도 제공합니다. 모든 데이터는 로컬 스토리지에 저장되며, 사용자가 직접 데이터를 초기화할 수 있습니다.

## 2. 컴플라이언스 요약

### 2.1 도박/사행성 방지

| 리스크 | 조치 |
|--------|------|
| **확률형 아이템** | 박스 오픈 시 확률 테이블을 UI에 명시 고지 (1등 0.01%, 2등 0.05%, 3등 0.30%, 디지털 99.64%) |
| **현금 유입 없음** | P(포인트)는 인앱 재화일 뿐, 현금 구매 기능 없음. 유료 충전 없음. |
| **참여비 없음** | 모든 퀴즈/승부차기/MVP 투표는 무료 참여. 포인트를 걸고 베팅하는 구조가 아님. |
| **결과 무관 보상** | 승부차기는 결과에 상관없이 고정 100P 지급. |
| **1일 제한** | 박스 오픈 1일 1회 제한. |
| **MVP 투표** | 참여비 0P, 정답 시에만 20P. 사행성 배팅 아님. |

### 2.2 저작권 회피

| 리스크 | 조치 |
|--------|------|
| **구단 엠블럼/로고** | 사용하지 않음. 유니폼 컬러 2색 블록 + 3자 약어로 대체 (예: `FC서울` → 빨강/검정 블록 + `SEO`). 상표권 침해 없음. |
| **리그 명칭** | "K리그"는 일반 명칭으로 사용. "EPL", "분데스리가", "라리가"는 상표권과 관련 없는 정보성 언급. |
| **문제 콘테스트** | 모든 축구 퀴즈 문제는 주관적인 지식 평가 문제로, 일반인의 축구 상식에서 파생. 평가지 원본을 복사하지 않음. |

### 2.3 개인정보 보호

| 항목 | 내용 |
|------|------|
| **수집 데이터** | 닉네임 (사용자 입력, 8자 제한), 퀴즈 점수, 스트릭, P 보유량 |
| **저장 위치** | 브라우저 localStorage (유저 디바이스 전용) |
| **서버 전송** | 없음 (MVP). 어떠한 사용자 데이터도 외부 서버로 전송되지 않음. |
| **제3자 제공** | 없음 |
| **데이터 삭제** | 설정 > 데이터 초기화 버튼으로 전체 데이터 삭제 가능 (localStorage.clear) |
| **쿠키** | 사용하지 않음 |

## 3. 간식박스 상세 분석

### 3.1 작동 방식

1. 사용자가 퀴즈/승부차기/응원으로 P 획득
2. 500P 보유 시 간식박스 앤뷰에서 박스 오픈 가능 (1일 1회)
3. 랜덤 알고리즘으로 등급 결정
4. 1~3등 실물 당첨 시 오픈카톡 문의 안내
5. 대표님이 수동 문의 대응 → 기프트나 현금으로 간치 발송

### 3.2 확률 및 비용

| 등급 | 확률 | 실비용 | 일당 예상 당정 수 (DAU 1,000) |
|------|------|------|------|
| 1등 | 0.01% | 3,000원 (편의 3,000권) | 0.1명 (10일에 1명) |
| 2등 | 0.05% | 1,000원 (육개장 기프티콘) | 0.5명 (2일에 11명) |
| 3등 | 0.30% | 500원 (추파춥스 교환권) | 3명 |
| 디지털 | 99.64% | 0원 (VAR권/복구P) | 996.4명 |

**월 실면서 총비용**: 약 55,500원 (일 1,850원 규모)

### 3.4 실물 정산 프로세스

```
[사용자 박스 오픈] → [1/2/3등 당첨]
                              │
                              ▼
                    [오픈카톡 링크로 문의]
                              │
                              ▼
                    [대표 직접 확인 → 기프트콘 발송]
```

## 4. 개인정보 처리방침 (전문)

### 4.1 수집하는 개인정보 항목

KICK QUIZ는 다음과 같은 개인정보를 수집합니다:

- **닉네임**: 사용자가 직접 입력한 8자 이내의 닉네임
- **앱 사용 데이터**: 퀀점 점수, P 보유량, 스트릭 일수, 쿠폰 함 보유 현황, 일일 챌린지 완료 여부

### 4.2 개인정보의 저장 및 보안

- 모든 데이터는 **사용자의 기기 내 localStorage**에만 저장됩니다.
- 서버로 전송되는 개인회는 없습니다.
- Toss 미니앱 웹뷰 환경에서 HTTPS로 접속됩니다.

### 4.3 개인정보의 제3자 제공

- **제공하지 않습니다.** 어떠한 사용자 데이터도 외부 서버, 제3자, 분석 도구에 공유되지 않습니다.

### 4.4 데이터 보유 기간

- 사용자가 앱 사용 중일 때만 보유됩니다.
- 사용자가 설정 메뉴에서 '데이터 초기화'를 선택하면 **즉시 모든 데이터가 영구 삭제**됩니다.
- Toss 미니앱을 제거하면 브라우저 환경에서 데이터가 자동으로 삭제됩니다.

### 4.5 사용자 권리

- **열람**: 설정 > 프로필에서 현재 데이터 조회 가능
- **삭제**: 설정 > 데이터 초기화 로 전체 삭제 가능
- **정정**: 프로필에서 닉네임 직접 수정 가능

### 4.6 문의

- 개인정보 관련 문의: 오픈카톡 채팅으로 대표에게 직접 문의
